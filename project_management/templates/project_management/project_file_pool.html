    About File Pool

    What is the File Pool?

    The File Pool is a centralized location where you can view and manage all files uploaded across all tasks in this project.

    Why are files missing?

    • Task to Subtask Conversion: When tasks are converted to subtasks, file references may be preserved but physical files might not transfer correctly
    • File System Changes: Files may have been moved or deleted from the server
    • Upload Failures: Previous upload attempts may have partially succeeded (database record created but file not saved)

    How to fix missing files:

    1. Locate the original file on your computer
    2. Navigate to the task and re-upload the file
    3. Delete the old missing file record
