Import Instructions
CSV File Format

The CSV file should contain the following columns:

  • id or transaction_id - Unique Stripe transaction ID
  • amount or Amount - Transaction amount
  • fee or Fee - Processing fee (optional)
  • currency or Currency - Currency code (e.g., USD, EUR, HKD)
  • status or Status - Transaction status
  • type or Type - Transaction type
  • created or Created (UTC) - Transaction date
  • customer_email - Customer email (optional)
  • description - Transaction description (optional)
How to Export from Stripe
  1. Login to your Stripe Dashboard
  2. Go to Payments â†’ All Payments
  3. Click the Export button
  4. Select date range and export format (CSV)
  5. Download the CSV file
  6. Upload it here
Import Notes
  • Duplicate Prevention: Transactions with the same Stripe ID will be skipped automatically
  • Safe Process: You can safely upload the same file multiple times
  • Account-Specific: Make sure to select the correct account for your CSV files
  • Precision: All amounts are stored in cents for accuracy
Note: If you see "All transactions already exist", it means these files have been imported before. This is normal and prevents duplicates. Only NEW transactions will be imported.
